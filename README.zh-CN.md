一个菜单栏日历应用，同时显示公历和农历。

## 安装

- **系统要求**: macOS 14 或更高版本
- **下载**: 从 [发布页面](https://github.com/sabercon/mini-calendar/releases/latest) 获取最新版本的 `MiniCalendar_x.x.x.dmg` 文件

## 使用

安装后启动应用程序，你会在菜单栏看到一个小小的日历图标。
点击图标即可打开日历。
